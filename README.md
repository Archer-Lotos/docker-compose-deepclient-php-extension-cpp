# Deep Client PHP extension implemented in C++

Deep Client - a way to connect your favourite language with Deep.

## Install/Build
```bash
docker-compose up
```

#### or
```bash
docker run -d -p 18080:80 --name my-php-container php:7.4.33-apache
```
